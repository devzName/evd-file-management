# EVD File Management

## Requirements

- Node.js v24.16.0
- npm 11.13.0

## Install

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open the local URL shown in the terminal, usually:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
